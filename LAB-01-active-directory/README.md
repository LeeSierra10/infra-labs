# LAB 01 – Active Directory Deployment

## Objetivo
Implementar un dominio de Active Directory en Windows Server.

## Tecnologías utilizadas
- Windows Server 2022
- Active Directory Domain Services
- VirtualBox

## Configuración realizada
- Instalación de Windows Server
- Promoción a Domain Controller
- Creación del dominio lab.local
- Creación de usuarios y OU

## Resultado
Dominio funcional con administración de identidades.
